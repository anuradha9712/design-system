### Usage

<br/>

#### Status Hint vs Badge

Status hint is used to display the status of a resource while a badge is used as a label to tag entities. 

![Status hint vs Badge](./images/statusHint-vs-Badge.png)
<Caption> Status hint vs Badge </Caption>

<br/>

#### Overflow Behavior in Status Hint

In case of overflow, the items will get truncated and can be viewed inside a tooltip on hover.
